### MAIN-Project3.java ###

MAIN-Project3.java is a Java method library for the implementation and execution 
of DNA Sequence Alignment, in both Dynamic Programming and Brute Method strategies.

## Installation

Use the command line prompts to call for class. Specify using:

Directory % javac MAIN-Project3.java


## Usage

# Initialization
Use command line prompt to first call for input file to be read into program.
Use following format: 

Directory % java "filename.txt"

Where filename.txt = name of the actual file in appropriate directory.
Example: 

Directory % java "Input-Example1.txt"


# returns "Proj3_Output.txt" Output file for Assigned input
Directory % java "Input-Example1.txt"
