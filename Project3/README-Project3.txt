### Proj3_CompiledCode_NicolasEscudero.java ###

Proj3_CompiledCode_NicolasEscudero.java is a Java method library for the implementation and execution 
of DNA Sequence Alignment, in both Dynamic Programming and Brute Method strategies.

## Installation

Use the command line prompts to call for class. Specify using:

Directory % javac Proj3_CompiledCode_NicolasEscudero.java


## Usage

# Initialization
Use command line prompt to first call for input file to be read into program.
Use following format: 

Directory % java "file.txt"

Where file.txt = name of the actual file in appropriate directory.
Example: 

Directory % java "DynamicLab2Input.txt"



# returns "Proj3_Output.txt" Output file for Assigned input
Directory % java "DynamicLab2Output.txt"

# returns "Proj3_Output.txt" Output file for one supplement input case
Directory % java "MyOutput - 40-80aa.txt"

# returns "Proj3_Output.txt" Output file for another supplement input case
Directory % java "MyOutput - S16large.txt"
